# MagicVideo

## Features

- CoreImage filters on realtime camera
- Record and save filtered video to photo library
- Orientations support
- Zoom
- Torch

## Support

Supports iOS 10 and above. Xcode 9.2 is required to build the latest code written in Swift 4.0.
